Projet MomoCar – Site de Vente de Voitures

Présentation

MomoCar est un site web de vente de voitures en ligne, conçu pour faciliter l'achat et la vente de véhicules neufs ou d'occasion. Il propose une interface claire, un système de connexion sécurisé, et des fonctionnalités complètes pour les utilisateurs comme pour les administrateurs.

Fonctionnalités principales

  • Affichage des véhicules disponibles avec détails et prix
  • Ajout, modification et suppression d'annonces (interface admin)
  • Formulaire d’inscription et d’authentification sécurisé
  • Gestion d’un compte utilisateur (profil, annonces)
  • Recherche et filtrage par marque, prix, type de carburant

Technologies utilisées

  • Frontend : HTML, CSS, JavaScript
  • Backend : PHP
  • Base de données : MySQL

Points forts

  • Architecture propre et claire avec MVC simplifié
  • Interface responsive et intuitive
  • Formulaires sécurisés avec vérification des données
  • Système complet d’administration de véhicules
