Projet CERIcar - Application de Covoiturage

Introduction

CERIcar est une application web développée dans le cadre de l’UE Programmation Web à l’Université d’Avignon. Ce projet a été réalisé avec le framework Yii, en respectant une architecture MVC claire et organisée. Il vise à offrir une plateforme fonctionnelle pour la gestion de trajets de covoiturage, avec une interface moderne et interactive.

Objectifs

Le but principal était de concevoir une plateforme fluide et intuitive, où les utilisateurs peuvent proposer et réserver des trajets. L’utilisation d’AJAX nous a permis de gérer les requêtes sans rechargement de page, améliorant l’expérience utilisateur (connexion, inscription, recherche dynamique, réservation...).

Architecture et Technologies

  • Framework Yii (PHP) : pour structurer le projet en MVC
  • AJAX et JavaScript : pour dynamiser l’interface
  • HTML/CSS : pour le rendu visuel responsive
  • MySQL : pour la gestion des données (utilisateurs, trajets, réservations)

Fonctionnalités développées

  • Inscription et connexion sécurisée des utilisateurs
  • Ajout et modification de trajets par les conducteurs
  • Recherche de trajets avec ou sans correspondance
  • Réservation de trajets avec mise à jour automatique des places disponibles
  • Interface de gestion des trajets réservés et proposés (profil utilisateur)
