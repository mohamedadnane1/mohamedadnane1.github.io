Présentation du Projet

Développée dans le cadre du cours "Application Mobile et Ergonomie" à l'Université d'Avignon, Amiibo Quiz est une application mobile Android développée en Kotlin. Elle permet de découvrir et tester ses connaissances sur les personnages Amiibo à travers un quiz interactif.

Fonctionnalités principales

  • Connexion à l’API publique https://www.amiiboapi.com/ pour récupérer les données en temps réel.
  • Choix des séries de jeu pour filtrer les Amiibos souhaités.
  • Stockage local des favoris avec Realm (ORM local).
  • Affichage dynamique des personnages avec images (Picasso) et nom dans un RecyclerView.
  • Jeu de quiz avec score, indice, sélection parmi plusieurs réponses et gestion des interactions.
  • Interface fluide avec ViewBinding, menu, navigation entre activités, et design ergonomique.

Technologies utilisées

Kotlin, Android Studio, API REST avec Retrofit, Realm Java SDK, ViewBinding, Picasso, Architecture MVC, RecyclerView, Intent, Menu, Swipe Gesture, etc.
